# Tests

Test files for VisioNova components.

## Files

- **test_binoculars.py** - Test BINOCULARS text detection accuracy
- **test_image_api.py** - Test image detection API endpoints

## Running Tests

```powershell
# Activate environment
.venv\Scripts\Activate.ps1

# Run specific test
python tests/test_image_api.py

# Or use pytest (if installed)
pytest tests/
```
